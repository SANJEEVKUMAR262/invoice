# Acadza Invoice System

A single-file invoice management system for Acadza.

## Features
- Employee dashboard
- PDF & DOCX invoice generation
- Gmail integration for sending invoices
- Monthly billing with TDS calculation

## Usage
Open `index.html` in a browser (must be served over HTTP, not file://)
